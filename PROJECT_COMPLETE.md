## 🔧 Troubleshooting

### App Won't Start?

```bash
# Clear and reinstall
cd attendance-management
rm -rf node_modules .next
npm install
npm run dev
```

### Port 3001 Already in Use?

```bash
# Change port in package.json
"dev": "next dev -p 3002"
```

### Build Errors?

```bash
# Type check
npm run type-check

# Lint check
npm run lint

# Rebuild
npm run build
```
